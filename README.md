A simple command-line task manager built with Python. (My first ever Python project)

Features:
 - Add, update, delete tasks
 - Mark tasks as todo, in-progress, or done
 - List tasks by status
 - JSON-based

Usage:
 python task_cli.py <command> [arguments]
